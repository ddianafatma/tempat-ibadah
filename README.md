# tempat-ibadah
Sebuah web map yang menampilkan sebaran tempat ibadah seperti masjid, gereja, vihara, pura dan klenteng di Kelurahan Bulusan Kecamatan Tembalang
Hal ini ditujukan agar masyarakat lebih mudah menemukan tempat ibadah yang diinginkan
Dalam hal perencanaan kota data ini dapat digunakan untuk evaluasi pemerataan tempat ibadah

Selamat Beribadah
